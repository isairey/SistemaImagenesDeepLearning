# 🌌 Acerca del proyecto

**Cluster-FK** es una plataforma de agrupamiento inteligente de imágenes que utiliza modelos de aprendizaje profundo para identificar similitudes visuales entre fotografías.

El sistema permite analizar grandes colecciones de imágenes y organizarlas automáticamente en grupos utilizando algoritmos de clustering no supervisado.

Su enfoque principal está orientado a:

* 🧠 Reconocimiento facial
* 🖼️ Agrupamiento de objetos
* 🤖 Inteligencia Artificial
* 📊 Análisis visual automatizado
* 🔍 Detección de similitudes

# 📊 Funcionamiento del clustering

## 🔍 Extracción de características

El sistema analiza cada imagen para generar representaciones numéricas mediante modelos de Deep Learning.

---

## 📈 Comparación de similitud

Las imágenes son comparadas utilizando vectores de características extraídos automáticamente.

---

## 🧠 Agrupamiento con DBSCAN

DBSCAN detecta grupos naturales de imágenes sin necesidad de conocer previamente la cantidad de categorías.

### Beneficios

* No requiere definir número de clusters
* Detecta ruido automáticamente
* Maneja grupos complejos
* Excelente para datos visuales
